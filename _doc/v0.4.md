# ml4js v0.4 規劃

## 目標

加入二元分類最小可用功能：

```ts
const { features, target } = makeClassification({ nSamples: 80, seed: 7 });
const model = new LogisticRegression();
model.fit(features, target);
const predictions = model.predict(features);
const accuracy = model.score(features, target);
```

## 功能範圍

1. `LogisticRegression`
   - binary classification only
   - `fit`, `predict`, `predictProba`, `score`
2. classification metrics
   - `accuracyScore`
   - `precisionScore`
   - `recallScore`
3. `makeClassification`
   - 支援樣本數、特徵數、seed
   - 產生可分的二元分類資料
4. classification plotting
   - 2D scatter
5. 測試
   - 模型、metrics、dataset、plot/system test

## 設計原則

1. 先只支援 `0/1` 標籤。
2. 先用穩定可讀的梯度下降，不先做複雜最佳化。
3. 每版都必須可測試、可建置、可 commit。
