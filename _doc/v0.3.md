# ml4js v0.3 規劃

## 目標

補上回歸教學與實驗常用工具：

```ts
const { features, target } = makeRegression({
  nSamples: 40,
  nFeatures: 1,
  noise: 0.1,
  seed: 7
});

const pipeline = new Pipeline(
  [{ name: 'poly', transformer: new PolynomialFeatures(2) }],
  new LinearRegression()
);
```

## 功能範圍

1. `PolynomialFeatures`
   - 支援 `degree >= 1`
   - 先做最常用的 per-feature polynomial expansion
2. `makeRegression`
   - 支援樣本數、特徵數、noise、seed
   - 每次同 seed 需可重現
3. plotting 補強
   - actual vs predicted 散點圖
4. 測試
   - `PolynomialFeatures` 單元測試
   - `makeRegression` 單元測試
   - 系統測試改用合成資料 + polynomial pipeline

## 設計原則

1. 先做教學最常用能力，不提前做完整 `sklearn.datasets`。
2. `PolynomialFeatures` 先不加入交互作用項，避免 API 膨脹。
3. 所有隨機資料都必須能以 seed 重現。
