# ml4js v0.6 規劃

## 目標

先補上可直接執行的 `examples/`，讓使用者能快速驗證目前套件功能。

## 功能範圍

1. `examples/regression.ts`
   - 合成回歸資料
   - pipeline + polynomial regression
   - 輸出 Plotly HTML
2. `examples/classification.ts`
   - 合成分類資料
   - logistic regression
   - 輸出 Plotly HTML
3. `examples/clustering.ts`
   - `makeBlobs` + `PCA` + `KMeans`
   - 輸出 Plotly HTML
4. 執行腳本
   - `npm run example:regression`
   - `npm run example:classification`
   - `npm run example:clustering`
5. 文件
   - `examples/README.md`

## 設計原則

1. 範例必須可直接執行，不要求使用者自行拼接程式。
2. 所有輸出都寫到相對路徑 `examples/output/`。
3. 範例內容要對應目前套件已完成功能，不展示尚未實作的 API。
