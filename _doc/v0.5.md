# ml4js v0.5 規劃

## 目標

加入無監督學習的最小可用功能：

```ts
const { features } = makeBlobs({ nSamples: 90, centers: 3, seed: 7 });
const projected = new PCA(2).fitTransform(features);
const labels = new KMeans(3).fit(projected).predict(projected);
```

## 功能範圍

1. `KMeans`
   - `fit`, `predict`
   - 支援 `nClusters`, `maxIterations`
2. `PCA`
   - `fit`, `transform`, `fitTransform`
   - 先支援常用 2D/低維投影
3. `makeBlobs`
   - 產生可分群資料
   - 支援 seed
4. unsupervised plotting
   - 2D cluster scatter
5. 測試
   - `KMeans` / `PCA` / `makeBlobs`
   - 系統測試跑完生成與視覺化

## 設計原則

1. 先做教學可用版本，不先追求大型資料效能。
2. `PCA` 先用 power iteration 求前幾個主成分。
3. `KMeans` 預設使用 deterministic 初始化，方便測試。
