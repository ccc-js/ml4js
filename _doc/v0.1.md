# ml4js v0.1 規劃

## 目標

建立一個可用的 `node.js + typescript` 套件 `ml4js`，提供類似 scikit-learn 的基礎使用方式：

```ts
const model = new LinearRegression();
model.fit(X, y);
const predictions = model.predict(X);
const score = model.score(X, y);
```

## 功能範圍

1. 套件骨架
   - `package.json`
   - `tsconfig.json`
   - `eslint` 與 `prettier`
   - `test.sh`
2. 核心 API
   - `Estimator` / `Transformer` / `Regressor`
   - sklearn 風格 `fit`, `predict`, `score`
3. 第一個模型
   - `LinearRegression`
4. 前處理
   - `StandardScaler`
5. 指標
   - `meanSquaredError`
   - `meanAbsoluteError`
   - `r2Score`
6. 視覺化
   - 用 Plotly 相容 figure 產生回歸結果圖
   - 可輸出 HTML 報表
7. 測試
   - 單元測試
   - 系統測試

## 設計原則

1. 全部使用相對路徑。
2. 保持模組切分清楚，避免單檔過大。
3. 先把最小可用版本做穩，再逐版擴充分類、管線、資料集與更多模型。
